Studio

Você pode selecionar uma das seguintes opções para editar o Portal SuiteCRM:

  • Editar um módulo. Selecione esta opção para editar o módulo de Acompanhamento de Bugs, Casos ou pistas.
  • Enviar uma folha de estilos. Selecione esta opção para editar a folha de estilos padrão do Portal SuiteCRM.
  • Sincronizar Portal. Selecione esta opção para sincronizar o Portal SuiteCRM com o servidor SuiteCRM.

Para editar um módulo, siga estes passos:

  1. Selecione o módulo que quer reconfigurar no Portal SuiteCRM.
  2. Selecione a View que quer editar.
    O editor do portal exibe uma barra de ferramentas a esquerda e a visão detalhada de campos na direita. Para alternar entre a Visualização de edição ou visualização em lista, selecione o link apropriado acima. A barra de ferramentas exibe os campos disponíveis que atualmente não estão na visualização de detalhes. Ela também exive um campo vazio e um campo em branco. Use o campo em branco para criar espaços vazios na view. Por exemplo, use um campo em branco após descrição para criar um espaço para digitar uma descrição detalhada. Use um campo vazio como um marcador que você poderá substituir com um nome de campo futuramente.
  3. Para adicionar um campo da barra de ferramentas para a visão detalhada, clique em Adicionar linha. Selecione o campo da barra de ferramentas e arraste-o para o campo vazio.
  4. Para remover um campo da visualização, selecione-o e arraste-o para a barra de ferramentas.
  5. Clique em Salvar para salvar suas alterações.

Para enviar uma folha de estilos, siga estes passos:

  1. Selecione a aba apropriada para Casos, Acompanhamento de Bugs ou Novidades.
  2. Clique em Explorar, navegue para o local da folha de estilos e selecione-a.
  3. Clique em Enviar para exibir a nova folha de estilos.

Para sincronizar o Portal SuiteCRM com o servidor SuiteCRM, siga estes passsos:

  1. Digite a URL para o portal que quer atualizar e clique em Ir.
  2. Digite seu usuário e senha e então clique em Iniciar sincronização.
